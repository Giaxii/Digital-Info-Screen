# THI GoGlobal - Digital Info Screen Mockups

## Step-by-Step Guide to View the Live Site

To view the interactive screens directly in your browser, follow these simple steps to activate GitHub Pages:

1. Open your repository on GitHub:
   👉 [https://github.com/Giaxii/Digital-Info-Screen](https://github.com/Giaxii/Digital-Info-Screen)

2. Click on the **Settings** tab at the top right of the repository menu.

3. On the left sidebar menu, look under the **Code and automation** section and click on **Pages**.

4. Under **Build and deployment** -> **Branch**:
   * Click the dropdown menu and select **`main`**.
   * Leave the directory folder path as **`/(root)`**.
   * Click the **Save** button.

5. Wait about **1 minute**, then refresh the page. Your live link will appear at the top of the Pages settings tab:
   👉 **`https://giaxii.github.io/Digital-Info-Screen/`**
